# Clay Competitor Deep Dive

For full detail, see [Notion: Competitor Research: Clay](https://www.notion.so/31a091bd863480a0b238dc26240df156)

## Overview
Clay (~$100M ARR) — workflow orchestration platform with 150+ data source aggregation. Human designs enrichment in a table UI.

**Our differentiation:** We are intelligence, not orchestration. Agent decides what to research. We own our data pipeline.

## Main Use Cases
- **Outbound prospecting** — Enrichment-driven outbound with multi-source data.
- **ABM with Webflow** — Account-based marketing with personalized web experiences.
- **CRM enrichment** — Automated data enrichment for existing CRM records.
- **Signal-based selling** — Trigger-based outreach on hiring, funding, tech stack changes.

| Company | Use Case | Results |

| **Sendoso** | Pipeline generation | $1M+ pipeline attributed |

## Implications for Eventual
- Clay proves the market for GTM data enrichment and intelligence is real and large.
- Clay's model is human-designed workflows in a table UI — labor-intensive, requires expertise.
- Eventual's agent-native approach is fundamentally different: the agent navigates and decides, not the human.
- Clay aggregates from 150+ sources but doesn't own the intelligence — we aim to own ours.
- Risk: Clay could add agent-native interfaces. Our advantage is purpose-built for agents from day one.
